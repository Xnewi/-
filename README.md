# SEUAutoReport
用于解决东南大学恼人的信息化疫情填报

## 使用方法
1. 下载AutoReport.py，在源码里填好你的一卡通和密码 / 在目录下创建一个名为C2P.txt的文件,以<一卡通> <密码> <个位数>为格式储存密码。比如:
> 213181234 123456 0  
> 213191234 123456 0  
> ...
2. 按下F5，享受一键起飞的快感。

## 其他
1. 你需要Python的基础环境来运行Python程序。
2. 因本人水平有限，该程序需要借助WebDriver使用。请在
[WebDriver下载地址](http://chromedriver.storage.googleapis.com/index.html)下载与本地GoogleChrome适配的版本。
